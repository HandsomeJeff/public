Backup/Restore your personal files


Make sure that the program is constantly running on your computer and have Internet access. Automated rebuilding process ensures reliable storage and constant availability of your data.

Safely share data with others (developing)


Create a private groups, configure file permissions, send secure links to your friends, freely and anonymously publish any data on the BitDust network.

Upload and hide critical data (coming soon)


Upload an remove the source file from your computer. Data in the BitDust network completely lose its integrity - we call that "data disintegration", read more.

Your data will be read, processed and distributed on a set of remote machines to be safely stored as encrypted fragments. All BitDust users, as well as you, keeps pieces of another's data on own computers - without any access to them.
